# TESRECO Intern Management Portal

## Project Overview
The TESRECO Intern Management Portal is a Flask-based web application designed to manage intern records efficiently. It provides a user-friendly interface along with REST APIs to perform CRUD operations on intern data using an SQLite database.

## Features

- Flask Web Application
- SQLite Database Integration
- REST API Development
- CRUD Operations (Create, Read, Update, Delete)
- Advanced Python Concepts
  - Generators
  - Iterators
  - Performance Calculation
- HTML, CSS, and Bootstrap Frontend
- Postman API Testing
- Clean Project Structure

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap
- REST API
- Postman

## REST APIs

- GET - Fetch Intern Details
- POST - Add New Intern
- PUT - Update Existing Intern
- DELETE - Remove Intern

## Database

- SQLite Database
- Persistent Data Storage
- Integrated with Flask

## API Testing
